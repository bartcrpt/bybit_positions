# Bybit Positions Monitor

Chrome-расширение для мониторинга позиций Bybit, баланса аккаунта, открытого PnL и закрытого PnL за текущий день прямо из popup-окна браузера.

Расширение рассчитано на быстрый read-only контроль портфеля. Оно не выставляет ордера, не закрывает позиции, не переводит средства и не отправляет API-ключи никуда, кроме официального API Bybit для подписанных запросов.

## Возможности

- Показывает equity, доступный баланс, занятую маржу, нереализованный PnL и закрытый PnL за сегодня.
- Отображает открытые `linear` и `inverse` позиции: размер, цену входа, mark price, плечо, ROI, маржу, цену ликвидации, TP/SL и время открытия.
- Поддерживает Unified Trading Account и Standard/Contract аккаунты.
- Позволяет выбрать автообновление каждые 5, 10, 15, 30, 60 секунд или только ручное обновление.
- Показывает badge у иконки расширения с количеством открытых позиций и цветом по PnL.
- Кэширует последний успешный снимок данных в `chrome.storage.session` и показывает его как устаревший, если Bybit временно недоступен.
- Может отправлять уведомления о позициях рядом с ликвидацией и о закрытых позициях.
- Поддерживает темную, светлую и системную тему.
- Добавляет пункт контекстного меню для быстрого открытия настроек.

## Безопасность

- Используйте только read-only API-ключ Bybit.
- Не включайте права на торговлю, переводы или вывод средств.
- Настройки и API-ключи хранятся в `chrome.storage.local` внутри профиля браузера.
- Расширение запрашивает доступ только к `https://api.bybit.com/*`.
- Подпись запросов создается локально через HMAC SHA-256 с помощью браузерного Web Crypto API.
- Проект не связан с Bybit и не является финансовой рекомендацией.

## Установка

1. Склонируйте или скачайте этот репозиторий.
2. Откройте `chrome://extensions` в Chrome или другом Chromium-браузере.
3. Включите `Developer mode` / `Режим разработчика`.
4. Нажмите `Load unpacked` / `Загрузить распакованное расширение`.
5. Выберите папку репозитория.
6. Откройте настройки расширения и добавьте read-only API-ключ Bybit.

## Настройка API-ключа Bybit

Создайте API-ключ в аккаунте Bybit с правами только на чтение данных аккаунта и позиций, которые нужно мониторить.

Рекомендуемые ограничения:

- только read-only доступ;
- IP restriction, если возможно;
- без прав на ордера, торговлю, переводы и вывод средств.

После сохранения ключа в настройках расширения нажмите `Проверить соединение`, чтобы убедиться, что Bybit принимает указанные credentials.

## Разработка

Расширение не требует сборщика. Исходные файлы загружаются браузером напрямую как ES-модули.

Базовый workflow:

1. Измените файлы в репозитории.
2. Откройте `chrome://extensions`.
3. Нажмите reload у `Bybit Positions Monitor`.
4. Заново откройте popup или страницу настроек.

Пример проверки синтаксиса:

```bash
node --check background/service-worker.js
node --check lib/bybit-api.js
node --check popup/popup.js
node --check options/options.js
```

В `package.json` также есть команда:

```bash
npm test
```

Она ожидает тесты в `tests/*.test.js`.

## Разрешения

Расширение использует следующие Chrome permissions:

- `storage` - хранение настроек, кэша состояния и списка предыдущих позиций.
- `alarms` - фоновое автообновление по расписанию.
- `notifications` - опциональные уведомления о ликвидации и закрытии позиций.
- `contextMenus` - быстрый переход к настройкам из контекстного меню иконки расширения.

Host permission:

- `https://api.bybit.com/*` - чтение данных аккаунта, позиций, closed PnL, тикеров и проверка API-ключа через Bybit REST V5.

## Текущие ограничения

- Валюта отображения зафиксирована как `USDT`.
- Расширение рассчитано на Chromium-браузеры с поддержкой Manifest V3.
- API-запросы идут только в production endpoint Bybit.
- Исторический PnL ограничен closed PnL за текущий день, который возвращает Bybit.
